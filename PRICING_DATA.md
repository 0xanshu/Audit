## OpenAI API

**Source:** https://openai.com/api/pricing
**Verified:** May 2026

Pricing is per-token, not subscription. Common reference points:

- GPT-4o: $2.50/1M input tokens, $10/1M output tokens
- GPT-4o-mini: $0.15/1M input tokens, $0.60/1M output tokens
- GPT-3.5-turbo: $0.50/1M input tokens, $1.50/1M output tokens

For audit purposes, teams are asked to report their monthly API bill directly.

## Verification Notes

Prices change frequently. Before using this data in a production audit, verify against the official pricing pages above. The audit engine flags discrepancies between self-reported spend and calculated MSRP — if a user reports paying significantly more than MSRP, it may indicate a stale price in this file rather than actual overpayment.
