# Unit Economics

## How It Makes Money

Audit is top-of-funnel lead gen. The free audit creates qualified leads — companies that have already quantified their AI overspend. Two paths:

**Path A — Consulting (year 1):** Charge 20% of first-year savings to implement recommendations.

**Path B — SaaS (year 2+):** Monthly subscription for ongoing spend monitoring. $99–$499/mo by team size.

---

## Path A Math

| Metric | Value |
|--------|-------|
| Avg savings per audit | $800/mo ($9,600/yr) |
| Conversion: audit → paid | 3% |
| Avg deal value | $1,920 |
| CAC | ~$50 |
| **LTV/CAC** | **38x** |

**$1M ARR:** 521 deals/yr → 10/mo → 333 audits/mo → **555 visitors/mo**. Very doable.

---

## Path B Math

| Metric | Value |
|--------|-------|
| Avg contract | $199/mo |
| Monthly churn | 5% |
| LTV | $3,980 |
| CAC | ~$150 |

**$1M ARR:** 419 customers. At 1% conversion → 41,900 audits total. Year 2+ play.

---

## Recommendation

Start with Path A for revenue + customer insight. Build Path B as the scalable product once the lead base exists. The audit tool is top-of-funnel for both.
